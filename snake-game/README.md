# Terminal Snake Game 🐍

A simple snake game that runs in your terminal, written in Python.

## Requirements

- Python 3.x
- `curses` module (included in Python standard library on Unix/macOS)

> **Windows users**: You may need to install `windows-curses`:
> ```
> pip install windows-curses
> ```

## How to Play

1. Run the game:
   ```bash
   python snake.py
   ```

2. Controls:
   - **Arrow keys** or **WASD** to move the snake
   - **Q** to quit the game

3. Objective:
   - Eat the food (`*`) to grow and increase your score
   - Avoid hitting the walls or yourself!

## Game Elements

- `#` - Snake body
- `*` - Food

## Scoring

- Each food eaten: **+10 points**

Enjoy the game!
